# SafeLaylar Browser Extension

## Installation Instructions

### Chrome

1. Download the `chrome` folder from this directory
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the `chrome` folder
5. The SafeLaylar extension will appear in your browser toolbar

### Firefox

1. Download the `firefox` folder from this directory
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on..."
4. Select any file in the `firefox` folder
5. The SafeLaylar extension will be loaded

For permanent installation:
1. Zip the contents of the `firefox` folder
2. Rename the zip file to have a `.xpi` extension
3. Drag and drop the `.xpi` file onto Firefox

### Safari

Safari extensions require additional setup through Xcode:

1. Download the `safari` folder from this directory
2. Open Xcode and create a new Safari Web Extension project
3. Replace the extension files with the SafeLaylar files
4. Build and run the extension
5. Enable the extension in Safari Preferences > Extensions

## Features

- **Real-time content filtering**: Automatically detects and blocks toxic content
- **Anti-grooming protection**: Identifies suspicious messaging patterns
- **Phishing protection**: Blocks known phishing sites and suspicious URLs
- **Scam alerts**: Warns about potential romance scams and fraud
- **Deepfake detection**: Alerts on potentially manipulated media
- **Privacy protection**: Blocks tracking and spyware attempts

## Configuration

Click the SafeLaylar icon in your browser toolbar to:
- View protection statistics
- Enable/disable specific protection features
- Access your full dashboard

## Privacy

SafeLaylar processes content locally on your device. We never send your browsing data to external servers. Only aggregate, anonymized statistics are collected to improve protection.
